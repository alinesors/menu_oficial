Portable Menu UI Export
=======================

This folder was generated by:
  dev_scripts/export_portable_menu_ui.ps1

How to import in another Godot project
--------------------------------------
1. Copy these folders/files into the target project root:
   - rust_extension
   - scenes
   - assets/menu_custom
   - addons/removed_template
   - resources/themes
   - override.cfg

2. Ensure the target project.godot contains:
   [autoload]
   AppConfig="*res://addons/removed_template/base/nodes/autoloads/app_config/app_config.tscn"
   SceneLoader="*res://addons/removed_template/base/nodes/autoloads/scene_loader/scene_loader.tscn"

3. Set run/main_scene as needed, e.g.:
   run/main_scene="res://scenes/opening/tela_inicial.tscn"

4. Build Rust extension in target project:
   - open terminal in rust_extension
   - run: cargo build

5. Confirm extension file points to the DLL:
   rust_extension/menu_ui_rust.gdextension
   windows.* = "res://rust_extension/bin/menu_ui_rust.dll"

6. If Godot shows stale cache errors:
   - close Godot
   - delete .godot/editor/filesystem_cache10
   - reopen the project
